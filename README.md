# Food For Eyes

Приложение для просмотра и взаимодействия с фотографиями из Unsplash. Позволяет просматривать ленту изображений, ставить лайки, просматривать детальную информацию о фото и управлять профилем пользователя.

## 🎯 Функциональность

- **Авторизация через Unsplash** - безопасный вход с использованием OAuth 2.0
- **Лента изображений** - бесконечная прокрутка фотографий с пагинацией
- **Взаимодействие с фото** - возможность ставить и убирать лайки
- **Просмотр детальной информации** - увеличение фото и масштабирование жестами
- **Профиль пользователя** - отображение информации о пользователе и аватара
- **Выход из аккаунта** - безопасный logout с очисткой данных

## 🛠 Технологии

- **Swift** - основной язык разработки
- **UIKit** - построение пользовательского интерфейса
- **MVP + Presenter** - архитектурный паттерн
- **WebKit** - для отображения веб-страницы авторизации
- **URLSession** - сетевые запросы
- **KeychainWrapper** - безопасное хранение токена
- **Kingfisher** - загрузка и кэширование изображений
- **ProgressHUD** - индикаторы загрузки
- **CoreData** - локальное хранение данных (опционально)

## 📋 Требования

- iOS 13.0+
- Xcode 14.0+
- Swift 5.0+
